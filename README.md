# Power BI Sales Dashboard 📊

## 📌 Project Overview
Designed and built an interactive sales dashboard in Power BI Desktop 
using the Superstore dataset. The dashboard features KPI cards, bar charts, 
a donut chart and a line chart with dynamic slicers for filtering.

## 🛠️ Tools Used
- Power BI Desktop
- Power Query (data transformation)
- DAX (basic measures)

## 📂 Dataset
- Source: Kaggle — Superstore Sales Dataset
- 9,800 rows of transactional sales data

## 📊 Dashboard Features

### KPI Cards
- Total Orders: 7,963
- Total Sales: $1.86M

### Visuals
| Visual | Type | Insight |
|---|---|---|
| Products That Generate Most Revenue | Bar Chart | Technology leads at $0.6M |
| How Sales Change Over Time | Line Chart | Sales peaked in 2017 |
| Sales by Region | Donut Chart | West leads at 31.87% |
| Top 10 Customers by Sales | Bar Chart | Sean Miller leads at $30K |

### Interactive Features
- Filter by **Category**
- Filter by **Region**
- Adjust **Date Range**

## 💡 Key Insights
- Technology drives the most revenue across all categories
- West region contributes 31.87% of total sales
- Sales peaked in 2017 before declining in 2018
- Top 10 customers all spend above $10,000
